## Terminology

### Twist Drill Bit

The most common type of drill bit. Some features to note:

<br/>

- Tip or Point: The sharp point at the end of the drill bit (1)
- Flutes: Helical grooves which allow chips to exit the drill (2)
- Shank: The area where the drill will be grasped by a chuck (3)

### Step Drill Bit

A good option for creating larger holes. Features include:

<br/>

- Steps: Typically at 1/8" increments, they allow for the different sizes of holes (1)
- Tip or Point: The sharp point at the end of the drill bit (2)
- Shank: Area where the drill bit will be grasped, typically have flat and angled surfaces (3)

### Keyless Chuck

Used to hold drill bits, the most common chuck on hand drills. Parts include:

<br/>

- Jaws: Typically three angled pieces that linearly move to grip the shank of a drill bit (1)
- Body: Rotating this will cause the jaws to clamp or retract, some ratchet to prevent loosening (2)

### Keyed Chuck

Found on most drill presses, requires a chuck key to properly clamp the jaws onto a bit's shank.

<br/>

- Body: Rotate by hand to prevent bit from escaping, then mate the chuck key with teeth to achieve proper tightness (1)
- Jaws: Same deal as a keyless chuck (2)
- Chuck Key: Used to properly tighten jaws, attaching it to the drill press ensures it will not be lost (3)

### Drill Press

This machine is perhaps the most important tool for making an Everybot. This machine will have more torque, allow for more downward pressure and create straighter holes than a hand drill.

<br/>

- Spindle: Holds the chuck and rotates when motor is powered (1)
- Chuck: Almost always will be keyed (2)
- Table: The surface that work will be held on, make sure not to drill into it (3)
- Table Adjustment Handle: Move to adjust height of table (4)
- Table Lock: Unlock to rotate table around column or adjust height, always lock before drilling (5)
- Feed Wheel: Used to lower the spindle (6)

## Hand Drilling

:::warning
Hand drilling may feel safer than operating machines but there is still a lot that can go wrong. Always wear safety glasses and make sure long hair is managed. 
<br/>
Sometimes you may find yourself placing a hand on the side your drill will be exiting. If you are unable to find a way around this make sure there is significant distance between the drill and your other hand. Experienced mentors have taken a drill to the hand from this, avoid it at all costs.
<br/>
Parts that are not held down properly can aggressively rotate or be flung. A step drill bit can kick the drill really hard, especially if using too high of a speed.
:::

<br/>

Drilling by hand is great for any lower precision task. Holes that require higher precision can be done with a hand drill but thru holes will require more focus and skill to make sure the drill bit is not angled when drilling into the 2nd surface.

### Drilling a hole

To start drilling a drill bit is needed, if the hole has a bit called out in the engineering drawing, try to use that one if possible. Start chucking the drill bit in the drill, make sure that the flutes are not in the chuck but make sure the shank is firmly in the chuck. Tighten the chuck. Make sure the tip of the drill bit is concentric with the chuck (make sure it is centered and doesn&rsquo;t wobble). If it does wobble it probably is engaged with only 2 of the 3 chuck jaws. 
<br/>

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '624.00px', height: '245.24px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_10.jpg")} width = "858.00" height = "335.50"  style={{ width: '624.00px', height: '1108.00px', marginLeft: '0.00px', marginTop: '-346.76px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<br />

When drilling anything it is important to make sure that your part cannot move. Clamping the part to a table or using vice is typically recommended. Make sure that you will not drill into anything you do not want to. With a table this can be done by ensuring the drilling location is hanging off the table or by putting a sacrificial surface underneath like wood. With a vise wood with a width smaller/equal to the part works but ideally [parallels](https://www.harborfreight.com/20-piece-parallel-set-789.html) will be used.

<br/>

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '482.00px', height: '299.00px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_11.jpg")} width = "661.38" height = "409.75"  style={{ width: '624.00px', height: '351.00px', marginLeft: '-142.00px', marginTop: '0.00px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<br/>

For most holes we will want the highest speed which will usually be number 2. For larger holes use the lowest number. Make sure that there is no torque limiting by setting the clutch to drill, typically identified by the drill bit icon. Finally the drill needs to be set to rotate correctly, to do this make sure that the forward button is pressed in. This can typically be found on the side of the drill, with forward being the arrow that points towards the drill bit.

If you have cutting fluid, apply it to the desired hole. Center the drill in the center punch dimple, pull the trigger and apply moderate pressure. Ensure the drill is straight as possible throughout the drilling.

<br/>

## Drill Press

A Drill Press is a machine that will help make significantly straighter holes and eliminate room for error if used correctly. It will generally take more time to drill individual holes but the time is well worth the accuracy you can achieve.

<br/>

<br/>

To start using a drill press, ensure that it is in a safe state, with the power unplugged or popping out the safety switch. Open up the chuck by rotating it such that the jaws of the chuck move away from the center. When the jaws are able to fit your bit, place it inside and hand tighten the chuck. Make sure that a good amount of the shank is within the chuck and that the jaws are not touching the flutes of the drill bit.

Next we will want to adjust the speed of the press. You may want to consult your manual but speeds between 2k and 3k should work well for #7 drill bits with the recommendation being closer to 3k. This speed should be fine for anything under &frac14;&rdquo;.

<p><br /> </p>

For drills that require manual adjustment, start by loosening the belt tension, do this by loosening the thumb screws near the motor. It may be held in place on both sides. Move the belt to the desired pulleys, it may be easiest to pull the belt in the desired direction while spinning the pulley by hand. When done make sure the belts are re-tensioned by fastening the motor away from the spindle, see your manual for specifics on the tension.

Now put your part into the drill press, we need to figure out how to fix it to the table. Clamping a plate directly to the table may prove challenging, especially in the center as the drill press may cause the part to flex into the hole, producing a non-straight hole.

<p><br /> </p>

<p><br /> <br /> </p>

This would not be the case for tube stock but make sure that you will not drill into the table. If you are at risk of drilling into your table, adding wood underneath will work well. In the case of box tube an even better solution is using a vise in combination with parallels or sacrificial tube stock.

Once the proper configuration is found we need to determine the height and position of the table. If the drill bit is not able to fully go through the part you or there is nothing preventing the bit from penetrating the surface of the table then it will need to be adjusted. The table can be locked into position via a locking mechanism. Undo that. The table&rsquo;s height can now be adjusted and it can be rotated about the column by hand. When adjusting the height always make sure that the final movement of the table is up, this will prevent backlash pushing the table down slightly. When done, lock the table in place.

<p><br /> </p>

Finally we can begin lining up the part, moving the part around until the drill bit is roughly centered over the center punch dimple. Lower the drill press onto the dimple paying special attention to the drill bit. If the drill bit deflect into the dimple, meaning the drill bit moves slightly at an angle to enter the hole then you are not centered. Continually peck with the drill bit until you find that the drill bit does not deflect. Clamp the part down in this location, ensuring that it does not change location.

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '482.50px', height: '452.34px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_29.jpg")} width = "661.38" height = "621.50"  style={{ width: '482.50px', height: '855.68px', marginLeft: '0.00px', marginTop: '-108.37px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

This process can be annoying but it is important to get it right. To make the process easier, if you find that you like the alignment in a certain direction you can place a clamp against the edge of the part. This will not lock the part down but you can press the part against the clamp to ensure you always have that measurement. This can be done twice before clamping the part down, ensuring it does not shift when clamping. 

<p><br /> </p>

<p><br /> </p>

Another good technique is to keep the part loose, peck until the drill bit does not deflect into the dimble. With the tip of the bit in the dimple, add a clamp, not lifting the bit until clamped.

<p><br /> </p>

When clamped, pull the bit down to the surface once more, ensure that the holes line up and that the workholding will not interfere with the handle or cutting process. At this point the drill press can be powered on to ensure that the drill bit is concentric with the collet. Look at the point of the drill bit, it should not oscillate left or right (a tiny amount may occur with cheaper presses, this can typically also be seen higher in the chain of hardware as well). Power off the press then ensure that the peck is still centered within the divet.

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '359.50px', height: '458.71px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_31.jpg")} width = "492.25" height = "629.75"  style={{ width: '359.50px', height: '640.03px', marginLeft: '0.00px', marginTop: '-8.36px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<p><br /> </p>

You are now ready to make the hole. Put some cutting fluid on the part. Power on the drill press lower into the part. You want to use a mild-moderate amount of downward force while drilling. If not enough force is applied it will cause excess heat and too much force could lead to other issues. If you are using sacrificial material you will be able to feel a difference in the material, when you reach it stop the operation.

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '269.00px', height: '519.00px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_32.jpg")} width = "368.50" height = "712.25"  style={{ width: '410.00px', height: '729.00px', marginLeft: '0.00px', marginTop: '-54.00px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<p><br /> </p>

When done make sure not handle the aluminum chips as they could be hot and will be sharp. These can be vacuumed up or be swept with a <span style={{ textDecoration: "underline"}}><a class="c10" href="https://www.google.com/url?q=https://www.homedepot.com/p/2-in-Flat-Chip-Brush-1500-2/100626098&amp;sa=D&amp;source=editors&amp;ust=1704348948669235&amp;usg=AOvVaw2NK-KzLrmHCcTeiE2sfC0i">chip brush</a></span>. The hole will need to be filed and deburred. Using a file on the backside to remove the protruding material and then finishing with a deburring tool is a good way to go about it. Manually twisting a step or regular drill bit in the hole may also work.

This would require you to flip the part then rotate it. Prioritize a good grip in the vise over doing this.

<h2><span style={{ fontSize: "16pt"}}>Match Drilling</span></h2>

<p><br /> </p>

Match drilling is an operation that goes through multiple assemblies at once. If given a pencil and told to make a near perfect circle, a stencil would make that significantly easier. Match drilling works the same for us, it is significantly easier to make 5 holes whose centers are 0.5&rdquo; inches apart on a line when we have a part with that pattern.

<p><br /> </p>

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '318.50px', height: '291.96px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_50.jpg")} width = "435.88" height = "398.75"  style={{ width: '318.50px', height: '565.49px', marginLeft: '0.00px', marginTop: '-131.88px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '277.50px', height: '290.84px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_51.jpg")} width = "380.88" height = "398.75"  style={{ width: '277.50px', height: '494.25px', marginLeft: '0.00px', marginTop: '-61.56px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<p><br /> </p>

This also works well when you need identical or mirrored parts.

<p><br /> </p>

Generally match drilling is best done with a drill press. When traveling long distances with a drill bit, it can be difficult to ensure a hole is straight the longer the distance gets. Additionally some drill bits may not be long enough to make it through all faces, requiring a 2nd drilling operation. It will take more time on the drill press but the increase in accuracy will help mitigate the compounding errors that can occur.

<p><br /> </p>

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '624.00px', height: '350.00px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_52.png")} width = "858.00" height = "479.88"  style={{ width: '624.00px', height: '357.00px', marginLeft: '0.00px', marginTop: '-7.00px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>

<p><br /> </p>

Here are some general tips for match drilling:

<p><br /> </p>

- If match drilling with an off the shelf item, try to use a drill press. Even though it seems quick and easy to use a hand drill, you can accidentally widen the template&rsquo;s hole. This is especially true for box tube.

- When using a template it is important to make sure it stays in the same location the whole time or when it matters. This can be done with solidly clamping so that no operations will result in the clamp needing to be moved. This can also be achieved by placing bolts in finished holes or using appropriately sized wood screws affixed to a plank.

- If match drilling occurs between two pieces of tube stock that must be identical, make sure to line them on one end only. Figure out what you will attach the tube stock to, this should be the edge that you attempt to align the parts at. It is best if this side is a factory edge, as filled edges may not be as easy to line up.

<h3><span style={{ fontSize: "14pt"}}>Bearing Plate With Tube Stock: </span></h3>

<p><br /> </p>

A drill press is highly recommended. As with all of these operations it is best to still scribe and possibly punch the location of every hole. Start by making the bearing hole, preferably with a pilot hole then finishing with the step bit. We do this because the bearing is the only hole that does not have a templated location.

If possible, also use clamps to prevent the bearing plate from moving, especially the rotation of the pattern is important.

<p><br /> </p>

<div style={{ textAlign: 'center'}}><div style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px'}}><span style={{overflow: 'hidden', display: 'inline-block', margin: '0.00px 0.00px', border: '0.00px solid #000000', transform: 'rotate(0.00rad) translateZ(0px)',  width: '218.00px', height: '339.00px'}}><Image autoLoad={"true"} img={require("/static/media/mechanical/drilling/image_54.jpg")} width = "298.38" height = "464.75"  style={{ width: '343.00px', height: '609.00px', marginLeft: '-66.00px', marginTop: '0.00px', transform: 'rotate(0.00rad) translateZ(0px)', maxWidth: "none"}}></Image></span></div></div>
